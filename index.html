Websites I've built

Pinnacle Portal

A hotel operations platform built with HTML, CSS, JavaScript, PHP and Bootstrap. Uses Leaflet for map views and Grafana for dashboards; the API was implemented in Delphi. Users can view hotels, drill into individual floors, and inspect devices (leak detectors, smoke detectors) per floor for real-time status and alerts.

Connected Sensors

Built using HTML, CSS, JavaScript, PHP and Bootstrap. Uses Leaflet for map views and Chart.js for charts; the API was implemented in Delphi. Platform to view locations of devices and inspect basic device stats (temperature, speed, etc.).

Halo Achievement Tracker

A comprehensive Halo achievement tracker that lists every in-game achievement and provides clear, step-by-step guides to unlock them. Built with Delphi TMS Web Core and backed by a PHP API.
